<h1 align="center"> Disk Analysis & Autopsy </h1>

<div align="center">

We analyze the E01 disk image using autopsy and look for useful artifacts related to the system compromise. We examine system users, files and directories,
network settings, installed programs, as well as an attempted exploitation of a vulnerability in the AES-CFB8 encryption implementation within the Netlogon Remote Protocol (MS-NRPC).
