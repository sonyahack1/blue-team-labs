This repository contains a collection of labs and analytical reports focused on `security threat investigation` and `analysis from a blue team perspective`.

It covers areas such as `OSINT`, `Digital Forensics`, `Memory Analysis`, `Reverse Engineering`, and more. In each writeup, I document the tools used,
techniques applied, and key findings derived from analyzing artifacts, systems, or datasets.

The purpose of this repository is to track my progress, improve analytical thinking, and build a structured knowledge base in blue team methodologies and
defensive security practices.
